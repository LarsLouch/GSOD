# O Último Brasileiro - Site Oficial

Site oficial do jogo "O Último Brasileiro" com imagens reais do jogo.

## 🚀 Como usar

1. Abra o arquivo `index.html` no seu navegador para visualizar o site
2. Ou use um servidor local:
   - Com Python 3: `python -m http.server`
   - Com Node.js: `npx serve`
   - Com VS Code: Use a extensão "Live Server"

## 📁 Estrutura

- `index.html` - Página principal do site
- `styles.css` - Estilos e design do site
- `script.js` - Funcionalidades JavaScript
- `README.md` - Este arquivo
- `imagens/` - Diretório de imagens
  - `capas/` - Imagens de capa (fundo, título, capa do jogo)
  - `prints/` - Screenshots do jogo (11 capturas)
  - `icons/` - Ícone do jogo

## 🎨 Características

- Design responsivo (funciona em desktop e mobile)
- Tema escuro com acentos em vermelho (#e94560)
- Imagens reais do jogo integradas
- Hero section com fundo e título do jogo
- Seção de capa do jogo em destaque
- Galeria de screenshots interativa com lightbox
- Animações suaves ao rolar a página
- Seções: Início, Capa, Sobre, Jogabilidade, Galeria e Contato
- Formulário de contato funcional (console.log)
- Navegação com scroll suave
- Efeito de parallax no hero section
- Favicon personalizado

## �🛠️ Personalização

### Cores
- Cor principal: `#e94560` (vermelho)
- Fundo escuro: `#1a1a2e` e `#0f0f1a`

Para alterar as cores, edite o arquivo `styles.css`.

### Conteúdo
Todo o conteúdo pode ser editado diretamente no arquivo `index.html`.

### Imagens
Para substituir as imagens, mantenha a mesma estrutura de pastas ou atualize os caminhos no HTML.

## 📝 Funcionalidades

- **Scroll suave**: Navegação fluida entre seções
- **Lightbox**: Clique nas screenshots para ver em tamanho grande
- **Animações**: Elementos aparecem suavemente ao rolar a página
- **Parallax**: Efeito de profundidade no hero section
- **Responsivo**: Adapta-se a diferentes tamanhos de tela
